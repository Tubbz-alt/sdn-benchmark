Interconnected OpenFlow Switches

Interconnected OpenFlow switches are described by a file whose path is specified in runtime/config/floodlight.properties as the property

    net.floodlightcontroller.security.SEFloodlight.interconnections.

Interconnection File EBNF

    IC_JSON     = '{' ic_nv_list '}'
                ;

    ic_nv_list  = ic_nv
                | ic_nv ',' ic_nv_list
                ;

    ic_nv       = alias_nv
                | conn_nv
                ;

    alias_nv    = '".aliases"' ':' alias_dict
                ;

    alias_dict  = '{' sw_nv_list '}'
                ;

    sw_nv_list  = sw_nv
                | sw_nv ',' sw_nv_list
                ;

    sw_nv       = '"' <switch name> '"' ':' '"' <switch ID> '"'
                ;

    conn_nv     = sw_port_str ':' sw_port_str
                ;

    sw_port_str = '"' sw_str '/' sw_port_number '"'
                ;

    sw_str      = <switch name>
                | <switch ID>
                ;

Example Interconnection JSON File

    {
        ".aliases": {"switch-1": "00:00:01",
                     "switch-2": "00:00:02",
                     "switch-3": "00:00:03"},
        "switch-1/1": "switch-2/1",
        "switch-2/2": "switch-3/1",
        "switch-3/2": "switch-1/2",
        "switch-3/3": "switch-1/3"}
    }
